Клановая система с GUI и поддержкой аддонов

## 📦 Установка
1. Скачайте `ZenexClans.jar`
2. Поместите в папку `plugins`
3. Перезапустите сервер

## 🎯 Команды
| Команда | Описание | Право |
|---------|----------|-------|
| `/clan create <название>` | Создать клан | `zenexclans.create` |
| `/clan invite <игрок>` | Пригласить | `zenexclans.invite` |
| `/clan join <клан>` | Вступить | `zenexclans.join` |
| `/clan leave` | Покинуть | `zenexclans.leave` |
| `/clan kick <игрок>` | Кикнуть | `zenexclans.kick` |
| `/clan gui` | Открыть GUI | `zenexclans.use` |

## 🔐 Права
| Право | Описание |
|-------|----------|
| `zenexclans.use` | Использовать команды |
| `zenexclans.create` | Создавать кланы |
| `zenexclans.invite` | Приглашать игроков |
| `zenexclans.join` | Вступать в кланы |
| `zenexclans.leave` | Покидать кланы |
| `zenexclans.kick` | Кикать игроков |
| `zenexclans.admin` | Административные права |
